# Pong
A simple pong implementation using Bevy. Written in Rust.


## To run
#### Prerequisites
The only prerequisite is having installed Rust + cargo, both can be installed at: [rustup.rs](https://rustup.rs/)

#### Build from source
```shell
git clone https://github.com/Georgefwm/pong
```
```shell
cd pong
```
```shell
cargo run
```
